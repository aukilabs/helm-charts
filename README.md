# Auki Helm chart repository

Based on [aukilabs/helm-charts](https://github.com/aukilabs/helm-charts)
and automatically published on `https://charts.aukiverse.com`

For more information on how to use the charts in this repository, see the official
documentation for [Relay (Hagall)](https://github.com/aukilabs/hagall/blob/main/docs/deployment.md#kubernetes)
and the [Domain server](https://github.com/aukilabs/domain-server/blob/main/docs/deployment.md#kubernetes)
or the specific README files in each chart folder.
